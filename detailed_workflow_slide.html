NASS Dome Chamber 3DGS Simulation Pipeline

A highly visual wireframe blueprint outlining the five key architectural phases required to capture, process, optimize, and serve photorealistic 3D chambers directly in standard browser viewports under 24 hours.

01

Reference Sourcing

Multi-angle architectural layout reference compilation.

▪
Acquire front, side, and top-down views of NASS Abuja dome.
▪
Verify specific wood paneling textures and green upholstery color values.
02

Path Interpolation

Higgsfield AI video sequence timeline composition.

▪
Stitch static 2D coordinates into a continuous camera walkthrough.
▪
Enforce strict camera path geometry rules to eliminate morphing artifacts.
03

Splat Extraction

3D Gaussian Splat point cloud mathematical construction.

▪
Epic RealityScan processes the video frames for depth alignment.
▪
Generates standard PLY point cloud baking in realistic scene lighting.
04

Splat Optimization

SupaSplat editor bounding box trimming and thinning.

▪
Trim background noise and limit active splat boundaries.
▪
Compress target file size under 15MB to guarantee mobile-device loading.
05

Dashboard Integration

Next.js WebGL dashboard runtime binding.

▪
Load optimized splats as interactive WebGL background panoramas.
▪
Overlay 10 player role slots and bind real-time state variable databases.
